Lucide icon wrapper — use for every glyph in Kubo UI. Never emoji.

```jsx
<Icon name="bot" />
<Icon name="plus" size={12} />
```

Available: x, plus, check, chevron-down, chevron-right, chevrons-up-down, search, house, bot, message-square, workflow, shield, sparkles, book-open, cpu, user, key, log-out, moon, clock, play, blocks, zap, activity, paperclip, arrow-up, file, image, panel-left, square, bell, ellipsis, pencil, trash, calendar.
