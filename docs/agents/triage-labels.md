# Triage Labels

As skills falam em cinco papéis canônicos de triagem. Este arquivo mapeia esses papéis para as
strings de label de fato usadas no issue tracker deste repo. Coluna direita = edite se você usar
outro vocabulário.

| Papel na skill    | Label no nosso tracker | Significado                                |
| ----------------- | ---------------------- | ------------------------------------------ |
| `needs-triage`    | `needs-triage`         | Mantenedor precisa avaliar esta issue      |
| `needs-info`      | `needs-info`           | Aguardando mais informação do reporter     |
| `ready-for-agent` | `ready-for-agent`      | Totalmente especificada, pronta p/ agente  |
| `ready-for-human` | `ready-for-human`      | Exige implementação humana                 |
| `wontfix`         | `wontfix`              | Não será tratada                           |

Quando uma skill menciona um papel (ex.: "aplique o label de triagem pronto-para-agente"), use a
string de label correspondente desta tabela.
